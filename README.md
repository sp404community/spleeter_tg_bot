# Spleeter Telegram Bot

Telegram бот для разделения аудио на дорожки + миди + питч калькулятор + bpm определитель + определитель тона. Сделал Viktor Demin [@vdiomin](https://t.me/vdiomin)

#### Что делает бот?
Бот разбивает трек (mp3, wav, ogg) на дорожки по инструментам (STEMS) и присылает их в формате wav.

#### Можно ли получать готовые STEMS в формате mp3 вместо wav?
Да - теперь там есть отдельная кнопка.

#### Есть ли ограничения на размер файлов и прочее?
В версии 2.1 больше нет ограничений на размер файлов.
Но мощности сервера не позволяют конвертировать длинные (более 5-7 минут) записи.
При возможности будет арендован более мощный сервер.

#### Мне нужно другое количество дорожек
@stemsplitter4bot - для разделения на 4 дорожки
@stemsplitter2bot - для разделения на 2 дорожки (голос/инструментал)

#### Почему так долго/быстро?
Файлы обрабатываются в порядке общей очереди - есть разумные ограничения на скорость обработки и отправки файлов.

#### Бот посылает мне ошибку размера файлов - что делать?
Попробуйте отправить файл заново.
Возможно, файл длинный по времени.

#### Бот на меня никак не реагирует!
1. Если бот не прислал сообщение о начале работы - попробуйте прислать другой фал/загрузить иначе и/или подождите.
2. Если бот прислал сообщение о начале работы/загрузки и ничего не прислал - пришлите файл заново чуть позже.

#### Почему некоторые файлы разделяются отлично, а некоторые нет?
Всё зависит от качества исходника.
Алгоритмы - несовершенны.
